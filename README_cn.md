# 图书馆智能助手系统

## 项目概述
这是一个完整的图书馆智能助手系统，包含上位机GUI应用、下位机嵌入式控制程序和串口通信模块，实现图书位置查询和自动导航功能。

## 系统工作流程
1. 用户通过GUI查询图书位置。
2. 系统解析图书所在区域（A-F）。
3. 通过串口发送导航指令给下位机。
4. 下位机控制小车移动到指定区域。

## 功能特点

### 上位机功能
- 图书位置查询与展示
- 用户友好的图形界面
- 实时对话交互
- 字体大小调整
- 热门话题快速访问
- 导航指令生成与发送

### 下位机功能
- 四电机PID精确控制
- 编码器速度反馈
- 前进/后退双PWM控制
- 电机自动校准
- 速度同步算法

### 通信功能
- 16进制指令传输
- 错误检测与处理
- 响应接收与解析

## 文件结构
图书馆-助手/  
├── library_core.py # 核心逻辑：图书数据处理与API交互

## 硬件要求

### 上位机
- Python 3.7+ 版本
- 支持串口通信的计算机

### 下位机
- STM32F4系列开发板
- 4个带编码器的直流电机
- 电机驱动模块
- USB转TTL串口模块
